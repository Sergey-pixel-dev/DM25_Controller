# Controller_STM32F446RET6

Контроллер высоковольтного импульсного блока (до 30 кВ) на STM32F446RET6. Оцифровка высокочастотного импульсного сигнала (длительность единицы микросекунд) встроенным АЦП методом стробоскопического преобразования.

## Функционал

- Мониторинг постоянных напряжений различных узлов блока
- Стробоскопическая оцифровка быстрых импульсов — эффективная частота дискретизации значительно превышает возможности встроенного АЦП
- Передача данных (Modbus-кадры и АЦП-кадры) на ПК по UART
- Связь по Modbus RTU

## Аппаратная часть

- MCU: STM32F446RET6
- Плата спаяна вручную на макетной плате

## Стек

C, STM32F446, CMSIS/HAL, ADC, DMA, таймеры, UART, Modbus RTU
